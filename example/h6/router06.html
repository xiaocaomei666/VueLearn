Vue-router的重定向-redirect

在路由配置表中配置路径，然后将component替换成redirect，重定向到你想跳转的路径下

{
      path:'/params/:newsId(\\d+)/:newsTitle',
      component:Params
    },{
      path:'/goback',
      redirect:'/'
    }

当重定向的路由需要传递参数时,我们只需要在redirect后边的参数里复制重定向路径的path参数就可以了

{  path:'/params/:newsId(\\d+)/:newsTitle',  component:Params},
{  path:'/goParams/:newsId(\\d+)/:newsTitle',  redirect:'/params/:newsId(\\d+)/:newsTitle'}
