alias别名的使用

实现类似重定向的效果

redirect和alias的区别
redirect：仔细观察URL，redirect是直接改变了url的值，把url变成了真实的path路径。
alias：URL路径没有别改变，这种情况更友好，让用户知道自己访问的路径，只是改变了router-view中的内容。

{
    path: '/hi1',
    component: Hi1,
    alias:'/jspang'
}

别名请不要用在path为’/’中，如下代码的别名是不起作用的。

{  path: '/',  component: Hello,  alias:'/home'}

别名也可以传递参数 :

alias: '/toHome/:user/:phone(\\d+)'
