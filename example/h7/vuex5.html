actions异步修改状态

actions是异步的改变state状态，而Mutations是同步改变状态

在store.js中声明actions
actions是可以调用Mutations里的方法的，在actions里调用add和reduce两个方法。
const actions ={    
 addAction(context){     
     context.commit('add',10)
    },     
reduceAction({commit}){        
  commit('reduce')
     } }

context：上下文对象，这里你可以理解称store本身。
{commit}：直接把commit对象传递过来，可以让方法体逻辑和代码更清晰明了。

引入和使用方式和mutation相似
